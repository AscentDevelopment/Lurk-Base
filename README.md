# Lurk

Lurk is an advanced packet based anti-cheat for beta 1.7.3 utilizing BetaProtocollib

## Wiki

| Command         | Permission        | Aliases               | Description                                        |
|-----------------|-------------------|-----------------------|----------------------------------------------------|
| `/lurk`         | `lurk.use`        | `none`                | Shows all available commands.                      |
| `/alerts`       | `lurk.alerts`     | `lurk alerts`         | Toggles alert messages.                            |
| `/banwave`      | `lurk.banwave`    | `lurk banwave`        | Shows all banwave subcommands.                     |
| `/bypass`       | `lurk.bypass`     | `lurk bypass, exempt` | Allows you to bypass all checks or certain checks. |
| `/info`         | `lurk.info`       | `lurk info`           | Shows useful information about a player.           |
| `/logs`         | `lurk.logs`       | `lurk logs`           | Show all the logs a player has violated.           |
| `/mitigation`   | `lurk.mitigation` | `lurk mitigation`     | Allows you to toggle mitigations on a player.      |
| `/lurk reload`  | `lurk.reload`     | `none`                | Reload All Lurk YML's.                             |
| `/lurk version` | `lurk.version`    | `lurk ver, lurk v`    | Shows Lurk current version.                        |

### Requirements

- Craftbukkit 1060 or [Poseidon Server](https://github.com/retromcorg/Project-Poseidon)
- [BetaProtocollib](https://github.com/DirtPowered/BetaProtocollib)
